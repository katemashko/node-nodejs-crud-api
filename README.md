## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<!-- get all users-->

GET http://localhost:3000/api/users

<!-- get user by id-->

GET http://localhost:3000/api/users/{userId}

<!-- create new user-->

POST http://localhost:3000/api/users

<!-- update user by id-->

PUT http://localhost:3000/api/users/{userId}

<!-- delete user by id-->

DELETE http://localhost:3000/api/users/{userId}
